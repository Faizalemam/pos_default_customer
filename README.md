# POS Default Customer – Odoo 16

Automatically set a default customer (Guest POS) when the POS session starts.
This module is fully compatible with Odoo 16 OWL POS.

## Features
- Auto-select default customer
- OWL onMounted hook
- Multi-branch support
- Lightweight & production-safe

## Installation
1. Copy module to custom addons
2. Update app list
3. Install module
4. Restart Odoo
